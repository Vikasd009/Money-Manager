Money Manager is a full-stack financial tracking web application developed as a Final Year Project.
The application enables users to securely register, log in, and manage their personal finances by recording income and expenses and viewing their financial activity on a responsive dashboard.

The backend is implemented using Spring Boot with RESTful APIs, and the frontend is developed using React.js with Tailwind CSS. The application uses MySQL for persistent storage of user and transaction data.
